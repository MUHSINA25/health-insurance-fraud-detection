# 🏥 Health Insurance Fraud Detection using Machine Learning

## 📌 Project Overview

Healthcare insurance fraud causes significant financial losses to insurance companies and affects the efficiency of healthcare systems. This project develops an **AI-based fraud detection system** that identifies potentially fraudulent health insurance claims using Machine Learning algorithms.

The system analyzes historical claim data, performs data preprocessing, applies machine learning models, evaluates performance, and predicts whether a claim is **Fraudulent or Genuine**.

---

## 🎯 Objectives

- Develop a machine learning model to detect fraudulent insurance claims.
- Analyze healthcare claim patterns using data analytics.
- Compare different classification algorithms.
- Improve fraud detection accuracy using optimized models.
- Provide a practical solution for real-time fraud prediction.

##  Project Workflow

### 1. Data Collection
The insurance claim dataset is collected containing customer, policy, and claim-related information.

### 2. Data Preprocessing

Steps performed:

- Handling missing values
- Removing unnecessary features
- Encoding categorical variables
- Feature scaling
- Preparing data for model training

---

### 3. Exploratory Data Analysis (EDA)

Performed analysis using:

- Distribution plots
- Correlation analysis
- Fraud vs genuine claim comparison
- Feature relationship visualization

---

### 4. Machine Learning Models

The following classification algorithms are applied:

- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine
- K-Nearest Neighbor
- Ensemble Models

---

## 📊 Model Evaluation

Models are evaluated using:

- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC Curve
- AUC Score

---

##  Results

The trained models successfully classify insurance claims into:

Genuine Claims  
Fraudulent Claims  

Ensemble-based models provide better performance by learning complex fraud patterns from multiple decision boundaries.
